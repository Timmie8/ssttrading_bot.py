# SST AI Quant Engine

Een Python-gebaseerde trading bot engine die de trendlogica van Pine Script (TradingView) nabootst. 
De bot analyseert meerdere tijdframe-trends op basis van de EMA 20.

## Functionaliteiten
* **Multi-Timeframe Analyse**: 5M, 15M, 30M, 1H, 4H, 1D.
* **Pine Script Logica**: Exacte berekening van de EMA 20 en trendbepaling.
* **Real-time Data**: Gevoed door de Finnhub API.
